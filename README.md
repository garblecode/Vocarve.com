# 声刻 Vocarve

**你的私人 AI 录音秘书 — 飞行模式也能用**

一款完全离线的实时语音转录 App。所有处理在手机本地完成——不联网、不上传、零订阅。打开即录，录完即得，你的声音永远不会离开你的设备。

## Features

- **完全离线** — 无需网络，飞行模式也能正常使用
- **实时转录** — 录音同时生成文字，录完即得
- **隐私至上** — 所有数据本地处理，不上传任何服务器
- **零订阅** — 一次购买，永久使用
- **99+ 语言** — 支持全球主流语言实时转录
